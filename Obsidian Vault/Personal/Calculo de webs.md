Para saber cuanto puedes cobrar solo piensa,
* Cuanto crees que vale cada hora de trabajo?
Si no sabes la respuesta a esa pregunta, puedes buscar en muchas paginas el ratio/hr de muchos trabajos pero pero pero... Para calculo puedes hacerlo asi:
* Calcula tus gastos mensuales ( los gastos esenciales )
* Calcula tus gastos innecesarios ( no esenciales ) 
* Calcula cuanto quieres ganar extra ( para lo que necesites )
Si por ejemplo tus gastos mensuales son 1000$ (USD) y quieres ganar extra otros 1000
entonces calculas, 2000 USD / 180 ( o 160 ) -> Ese es tu ratio por hora , por ejemplo 12.5 en el caso de 160
Ahora, sabiendo que cada hora tuya la valuas en 12.5, tienes que calcular cuanto te tardarias en hacer la web
Ahora, este mismo ejemplo lo puedes hacer para todo tu equipo pero tienes una ventaja:
* La gente que trabaja para ti ya te va a decir cuanto espera ganar
* Tu tiense la ultima palabra de cuanto pagarles por lo que puedes sacar un ratio extra de lo que ya cobran ( como lo hacen todas las paginas tercera )
Y luego calculas tiempos de trabajo * lo pagos a cada uno  ( y un extra si es que buscas sacar mas, quien sabe )
